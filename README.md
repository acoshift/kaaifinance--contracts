# Kaai Finance Contracts

## Compile

```shell
npx hardhat compile
```

## Test ?

```shell
npx hardhat test
```

## Deploy

```shell
npx hardhat run ./scripts/deploy.ts --network rei
```

## Verify

```shell
npx hardhat verify CONTRACT_ADDRESS --network rei RECEIVER SIGNER
```
